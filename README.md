# Go Barber
Sistema simples para agendar serviços de corte de cabelo com acesso para usuário e cabeleireiro.

## Desenvolvido em
NodeJS + Postgresql

## Start 
npm start
Spript em package.json ("start": "nodemon src/index.js")
